# prabhuthangavel-labs

Hands-on labs for [prabhuthangavel.com](https://www.prabhuthangavel.com) courses. Each folder contains a self-contained lab environment that runs in GitHub Codespaces — zero setup required.

## Available Labs

| Course | Status |
|--------|--------|
| [Terraform](./terraform/) | ✅ Available |
| Kubernetes | 🔜 Coming soon |
| Docker | 🔜 Coming soon |
| CI/CD | 🔜 Coming soon |

## How It Works

1. Pick a course folder
2. Click **"Open in Codespaces"**
3. Everything is pre-installed — start coding immediately
