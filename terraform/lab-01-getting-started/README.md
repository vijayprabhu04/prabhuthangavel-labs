# Lab 01: Getting Started with Terraform

| Sub-Lab | What You'll Do |
|---------|---------------|
| `01-installing-terraform/` | Install Terraform from scratch on a bare machine |
| `02-your-first-s3-bucket/` | Create, modify, and destroy an S3 bucket |
| `03-understanding-state/` | Explore the `.tfstate` file and state commands |

## Getting Started

```bash
cd terraform/lab-01-getting-started/02-your-first-s3-bucket
terraform init
```

> 💡 Sub-lab 01 uses a **bare Codespace** (no Terraform pre-installed). Sub-labs 02 & 03 use the full lab environment with LocalStack.
