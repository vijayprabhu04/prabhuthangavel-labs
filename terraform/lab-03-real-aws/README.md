# Lab 03: Real AWS Infrastructure

**Prerequisites:** AWS account with IAM user (programmatic access). Run `aws configure` first.

| Sub-Lab | What You'll Do |
|---------|---------------|
| `01-aws-provider-setup/` | Configure credentials and create a test resource |
| `02-vpc-and-security-groups/` | Build VPC, subnet, IGW, security groups, and EC2 |
| `03-remote-state/` | Store state in S3 for team collaboration |

> ⚠️ This lab creates **real AWS resources**. Always `terraform destroy` when finished.
